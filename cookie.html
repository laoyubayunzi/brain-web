Cookie政策

最后更新日期：2024年7月1日

SAU脑机与人工智能俱乐部（以下简称"我们"）使用Cookie和类似技术来为您提供更好的用户体验，并帮助我们改进网站。本政策旨在向您说明我们如何使用这些技术，以及您如何控制它们。

通过访问或使用我们的网站，您同意我们按照本政策的规定使用Cookie和类似技术。如果您不同意本政策的任何内容，请立即停止访问或使用我们的网站，或根据本政策的说明禁用Cookie。

1. 什么是Cookie？

Cookie是一种小型文本文件，当您访问网站时，网站可能会存储在您的设备上。Cookie通常包含一个标识符、网站名称和一些号码及字符。Cookie可以帮助网站记住您的偏好设置、分析您的使用模式，从而提供更好的用户体验。

除了传统的Cookie，我们还可能使用其他类似技术，如网络信标（web beacons）、像素标签（pixel tags）、本地存储（local storage）等。这些技术在功能上类似于Cookie，用于收集和存储用户信息。

2. 我们使用的Cookie类型

我们使用不同类型的Cookie来实现不同的功能。根据Cookie的用途和持续时间，我们使用的Cookie主要分为以下几类：

  • 必要Cookie（Strictly Necessary Cookies）： 这类Cookie对于网站的基本功能是必需的，例如记住用户的登录状态、保持会话状态、防止欺诈等。如果禁用这些Cookie，网站的部分功能可能无法正常运行。这些Cookie通常在用户访问网站时自动设置，不需要用户同意。
  • 功能性Cookie（Functional Cookies）： 这类Cookie用于增强网站的功能，例如记住用户的偏好设置、语言选择、字体大小等。这些Cookie可以提高用户体验，但即使禁用，网站的核心功能仍然可以使用。
  • 分析Cookie（Analytical Cookies）： 这类Cookie用于收集有关网站使用情况的统计信息，帮助我们了解用户如何使用网站，从而改进网站。这些Cookie收集的信息通常是匿名的，不会与个人身份信息关联。我们使用的分析工具可能包括Google Analytics等第三方服务。
  • 营销Cookie（Marketing Cookies）： 这类Cookie用于提供个性化的广告内容，追踪广告效果。它们可能会记录您访问过的网站，以及您如何与网站互动，从而提供与您兴趣相关的广告。这些Cookie通常由第三方广告提供商设置。

3. 我们如何使用Cookie？

我们使用Cookie和类似技术的主要目的包括：

  • 确保网站正常运行：维护用户会话、防止欺诈等。
  • 改善用户体验：记住用户的偏好设置、语言选择等。
  • 分析网站使用情况：了解用户如何使用网站，识别热门页面，检测错误等。
  • 优化网站性能：根据用户使用模式改进网站功能和性能。
  • 提供个性化内容：根据用户兴趣提供相关的内容和广告。
  • 进行市场研究：了解用户需求和行为，开发新功能和服务。

4. 第三方Cookie

除了我们自己的Cookie外，我们的网站还可能使用第三方Cookie。这些Cookie由第三方服务提供商设置，用于提供服务，如分析、广告和社交分享等。

我们使用的第三方服务提供商可能包括：

  • 分析服务提供商（如Google Analytics）：用于收集网站使用数据，帮助我们分析和改进网站。
  • 社交媒体平台（如微信、微博等）：用于社交分享功能和身份验证。
  • 广告服务提供商：用于提供个性化广告和广告效果分析。

第三方Cookie的使用受第三方服务提供商的隐私政策和Cookie政策的约束。我们不对第三方的Cookie使用行为负责，建议您查看这些第三方的隐私政策，了解他们如何收集和使用您的信息。

5. 如何管理Cookie？

您可以通过多种方式管理Cookie，包括浏览器设置、Cookie同意管理工具等。请注意，禁用某些Cookie可能会影响网站的部分功能。

5.1 通过浏览器设置管理Cookie

大多数网络浏览器允许您管理Cookie设置，包括阻止所有Cookie、仅阻止第三方Cookie、删除已存在的Cookie等。以下是一些常见浏览器的Cookie设置指南：

  • Google Chrome：点击菜单图标 → 设置 → 隐私和安全性 → Cookie和其他网站数据。
  • Mozilla Firefox：点击菜单图标 → 设置 → 隐私与安全性 → Cookie和站点数据。
  • Apple Safari：点击Safari → 偏好设置 → 隐私 → Cookie和网站数据。
  • Microsoft Edge：点击菜单图标 → 设置 → 隐私、搜索和服务 → Cookie和站点权限。

请注意，不同浏览器的设置界面和选项可能有所不同。有关更详细的指导，请参考您使用的浏览器的帮助文档。

5.2 管理分析Cookie和营销Cookie

对于分析Cookie和营销Cookie，您还可以通过以下方式进行管理：

  • Google Analytics Opt-out Browser Add-on：您可以安装Google Analytics Opt-out Browser Add-on来阻止Google Analytics收集您的网站访问数据。
  • 网络广告行业自律机制：您可以通过各地区的网络广告行业自律机制（如数字广告联盟的选择退出工具）管理营销Cookie。

6. Cookie的存储期限

根据Cookie的类型和用途，Cookie的存储期限可能有所不同：

  • 会话Cookie（Session Cookies）：这类Cookie仅在您的浏览器会话期间存在，当您关闭浏览器时会被删除。会话Cookie主要用于维护用户在当前会话中的状态和信息。
  • 持久Cookie（Persistent Cookies）：这类Cookie会在您关闭浏览器后继续存在于您的设备上，直到达到设定的过期日期或被手动删除。持久Cookie用于记住用户的偏好设置和其他信息，以便在后续访问时提供更好的用户体验。

我们会根据Cookie的用途和必要性，设置合理的存储期限，通常不会超过实现其目的所需的时间。

7. Cookie政策的更新

我们可能会不时更新本Cookie政策，以反映我们的Cookie使用情况的变化、技术的发展或法律法规的要求。当我们对Cookie政策做出重大变更时，我们将通过在网站上发布修订后的Cookie政策并更新"最后更新日期"来通知您。

我们鼓励您定期查看本Cookie政策。您继续使用我们的网站将被视为您接受修订后的Cookie政策。
